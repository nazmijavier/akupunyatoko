Sistem Box / Paket

Aturan Mingguan

Klik hari untuk set status default tiap minggu (Tersedia → Perlu konfirmasi → Penuh). Berlaku otomatis ke semua tanggal yang belum kamu atur manual di kalender bawah.

Ketersediaan Tanggal

Klik tanggal untuk override status tertentu: Tersedia → Perlu konfirmasi → Penuh → balik ngikut aturan mingguan. Tanggal Penuh nggak bisa dipilih pembeli. Tanggal ber-titik = di-override manual.
